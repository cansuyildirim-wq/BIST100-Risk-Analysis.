# BIST100-Risk-Analysis.
BIST 100 verisi üzerinden hareketli ortalamalar, RSI, volatilite ve normallik analizi yapan Python çalışması.
BIST 100 Technical & Normality AnalysisBu proje, BIST 100 endeks verilerini kullanarak teknik analiz göstergeleri ve istatistiksel risk modelleri oluşturur.Analiz KapsamıMoving Averages (MA50 - MA200): Uzun ve kısa vadeli trend analizi.Volatility: Yıllıklandırılmış oynaklık hesaplaması.Normality Testing: Getirilerin normal dağılıma uygunluğunun $p$-value üzerinden testi.Q-Q Plot: Ekstrem risklerin (fat tails) görselleştirilmesi.
